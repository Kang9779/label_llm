# 大模型对话标注工具

基于Flask的Web应用，用于标注和管理大模型对话数据。

## 项目结构

## 功能特点

- 批量上传对话JSON文件
- 任务列表管理（支持分页显示）
- 对话内容的编辑和删除
- 左右布局的对话界面
- 自动保存标注状态
- 支持查看已完成标注

## 对话数据格式
```json
[
    {
        "role": "user",
        "content": "用户的对话内容"
    },
    {
        "role": "assistant",
        "content": "助手的回复内容"
    }
]
```

## 开发环境

- Python 3.6+
- Flask 2.0.1
- 现代浏览器（Chrome/Firefox/Safari）
